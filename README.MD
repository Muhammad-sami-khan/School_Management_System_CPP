
# School Management System

## Overview

The School Management System is a comprehensive software solution designed to facilitate efficient management of various administrative tasks within educational institutions. This system is built using C++ programming language and offers a user-friendly interface for managing students, teachers, staff, accounts, and rooms.

## Features

### Student Management
- **Add/Edit Students**: Easily add new students to the system or edit existing student details such as name, father's name, class, section, fee, and attendance records.
- **View Student Stats**: View detailed statistics for each student including attendance, fee status, and other relevant information.

### Staff Management
- **Add/Edit Staff Members**: Manage the details of teaching and non-teaching staff by adding new staff members or editing existing staff information.
- **View Staff Stats**: Access comprehensive statistics for each staff member including salary, duty, attendance, and performance metrics.

### Teacher Management
- **Add/Edit Teachers**: Add new teachers or modify existing teacher information including name, course taught, salary, and attendance records.
- **View Teacher Stats**: View detailed statistics for each teacher such as course taught, salary, attendance, and performance metrics.

### Accounts Management
- **Expense Tracking**: Track monthly expenses incurred by the institution for various purposes such as maintenance, utilities, and supplies.
- **Profit Monitoring**: Monitor monthly profits generated by the institution through various sources such as tuition fees, stationery sales, and external ventures.

### Room Management
- **Check Room Availability**: View available classrooms and facilities within the institution, including class numbers and sections.

## Installation and Usage

1. **Clone the Repository**: Clone this repository to your local machine using the following command:
    ```
    git clone <repository-url>
    ```

2. **Compile the Code**: Navigate to the project directory and compile the code using a compatible C++ compiler:
    ```
    g++ main.cpp -o school_management_system
    ```

3. **Run the Program**: Execute the compiled executable to run the School Management System:
    ```
    ./school_management_system
    ```

4. **Navigate Through Menus**: Use the intuitive menus provided to navigate through different functionalities and manage school data efficiently.

## Dependencies

- **C++ Compiler**: The program is written in C++ and requires a compatible compiler such as g++.
